# CRE-Python-Solution

Python solutions for Chemical Reaction Engineering (CRE) Assingments covering chemical kinetics, batch reactor data analysis, reactor sizing, recycle reactors, and non-ideal flow systems.

## Topics Covered

- Introduction to Chemical Kinetics
- Interpretation of Batch Reactor Data
- Sizing of Reactors and Series Arrangement of Reactors
- Recycle Reactors
- Temperature and Pressure Effects
- Non-Ideal Flow Systems

## Tools Used

- Python
- NumPy
- SciPy
- Matplotlib
- Jupyter Notebook

## Repository Structure

Assignment-1 → Introduction to Chemical Kinetics

Assignment-2 → Interpretation of Batch Reactor Data

Assignment-3 → Reactor Sizing and Series Arrangement

Assignment-4 → Recycle Reactors

Assignment-5 → Temperature, Pressure Effects & Non-Ideal Flow
